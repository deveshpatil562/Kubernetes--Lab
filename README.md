## ⚙️ Setup (Quick Start)

Install kubectl:

[
curl -LO "https://dl.k8s.io/release/$(curl -L -s https://dl.k8s.io/release/stable.txt)/bin/linux/amd64/kubectl"
chmod +x kubectl
sudo mv kubectl /usr/local/bin/
]

Create cluster using kind:

[
kind create cluster --name devops-cluster
]

Verify:

[ kubectl get nodes ]
